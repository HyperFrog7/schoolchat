✦ What's New
SchoolChat V2 🎉
The biggest update yet — AI assistant, links board, search, slow mode, and much more.
🔍 Search & Navigation
🔍
Message Search New
Press Ctrl+F or click the search icon in the topbar to search messages in the current channel. Navigate results with ▲▼ arrows.
⬆️
Jump to Unread New
When you have unread messages above, a "Jump to Unread" button appears at the bottom of the chat. Click it to scroll directly to where you left off.
🛡️ Moderation
⏱️
Slow Mode New
Set a per-channel cooldown so users can only send messages every 5s, 10s, 30s, 1 min, or 5 min. A countdown timer shows in the input bar. Configure in Settings → Moderation.
📢
Announcement Channels New
Lock any channel to announcement-only mode — only the owner can post, everyone else can read and react. Set in Settings → Moderation.
🖼️ Media
🔎
Image Lightbox New
Click any image in chat to open it in a full-screen lightbox. Press Escape or click outside to close.
👤 Profiles
📅
Join Date on Profile New
User profiles now show when someone joined SchoolChat.
🕐
Last Online on Profiles New
Profiles now show when someone was last online — "Last seen 5m ago", "Last seen yesterday", etc. Updates live.
⌨️ Keyboard Shortcuts
⌨️
Keyboard Shortcuts New
Ctrl+F opens message search. Escape closes any open modal, search bar, or lightbox instantly.
✦ AI Assistant
✦
AI Assistant New
Click ✦ in the message bar to open the SchoolChat AI panel. Ask homework questions, get explanations, brainstorm ideas — it streams responses in real time and remembers your conversation history.
🔗 Links Board
🔗
Links Channel New
The #links channel is now a dedicated link board with categories. The owner can add/remove categories and links. Everyone can browse and click them.
🗳️
Polls New
Create polls with custom options, presets like Yes/No, and live vote tracking. Multiple choice supported.
😄
Reactions New
React to any message with emoji. Hover a message to find the react button.
🎬
GIF Support New
Search and send GIFs directly in chat using the GIF button in the toolbar.
📎
File Uploads New
Share images and files directly in chat. Images preview inline.
🔍 Emoji
🔎
Searchable Emoji Picker Improved
800+ emoji with names. Type what you're looking for — "fire", "crying", "thumbs up" — and it finds it instantly.
✨
Custom Server Emoji New
Admins can upload custom emoji that everyone can use. Type :name: or pick them from the emoji picker.
📁 Organization
📂
Channel Categories New
Channels are now organized into collapsible categories. Click a category to collapse or expand it.
🖱️
Drag & Drop Channels New
Drag channels to reorder them or drop them into a different category. Admins only.
👥 Social
💬
Group Chats New
Start a private group chat with multiple people from the DMs panel.
🟢
Presence & Profiles New
See who's online, idle, or offline in real time. Click anyone's name to view their profile.
🛡️ Roles & Moderation
🎖️
Role System New
Admins can create custom roles with colors, assign them to users, and control who can post in which channels.
🥷
Tab Cloaking New
Disguise the tab as something else with a custom title and favicon. Hit the panic button to instantly switch.
✨ Custom Emoji
🖼️
Upload Server Emoji New
Admins can upload custom emoji (PNG, GIF, JPG) that appear in the picker for everyone. Click ✨ in the toolbar to manage them.
⌨️
:shortcode: Support New
Type :name: in any message to use a custom emoji. They render as images inline — just like Discord.
😄
Custom Emoji Reactions New
React to messages with custom server emoji. They show as images in the reaction chips below messages.
🛡️ Moderation
⚠️
3 Warnings Limit New
Users are shown how many warnings they have remaining out of 3. At 3 warnings they're notified that further violations may result in a ban.
⏱️
Temporary Bans New
Admins can issue temp bans with a duration of 1 hour, 6 hours, 1 day, 3 days, or 7 days. Access is automatically restored when the ban expires.
🎨 Visual Polish
✨
Logo Gradient New
The "Chat" part of the SchoolChat logo now has a blue gradient — from brand blue to a lighter sky blue. Applied across the loading screen, auth screen, sidebar, and favicon.
🟢
Custom Statuses New
Set a custom status with an emoji and text — like "📚 Studying" or "🎮 Gaming". Click your status dot in the sidebar to set it. Click the emoji button to pick any emoji from the full picker. Custom statuses show in the members panel and on profiles.
📐
DeltaMath Cloak New
Added DeltaMath Student Application as a tab cloak preset.
🚩 Reporting
🚩
Report Button New
Report messages, users, or channels with one click. Hover any message to see the 🚩 button in the action bar, hover a user in the members panel, or hover a channel in the sidebar. Choose a reason and add optional details.
📋
Reports Panel (Owner) New
Owners see a 📋 button in the sidebar with a red badge showing open report count. Click it to view all reports, filter by type or status, mark reports resolved, jump to reported users, or delete reports.
🎨 Themes
🌑
Theme Presets New
8 built-in themes — Default, Midnight, Forest, Rose, Sunset, Ocean, Light, and Soft. Switch instantly from Settings → Appearance.
🖌️
Custom Colors New
Pick your own background, surface, and accent colors with custom color pickers. Your theme saves and restores automatically on every visit.
🔤
Font Picker New
Choose from DM Sans, Inter, Syne, Georgia, or Mono. Font preference is saved per device.
🔧 Bug Fixes
🗑️
Message Deletion Ghost Fixed
Deleting a message no longer causes a random old message to reappear at the bottom of chat. Fixed by tracking the newest rendered timestamp and ignoring window-shift artifacts from Firestore's limitToLast re-evaluation.
💾
Role Permissions Not Saving Fixed
Warn, Mute, Ban, and @everyone permissions now correctly save and load when editing a role.
🔎
Emoji Search by Name Improved
The emoji picker now searches by actual emoji names — type "thumbs", "fire", or "crying" to find what you need.
🔒
Custom Emoji Permissions Fixed
Fixed a Firestore permission error that prevented custom emoji from loading for non-admin users. Requires updated Firestore rules.
✨
Custom Emoji Invisible to Non-Admins Fixed
Fixed a crash in the emoji picker that prevented the ✨ custom tab from appearing for non-admin users. Also fixed a race condition where the tab wouldn't show if emoji loaded after the picker was built.
⌨️
Emoji Picker Position & :autocomplete: Fixed
Emoji picker now opens above the button instead of covering it. Type :name: to get a searchable dropdown that auto-inserts the emoji — works for both standard and custom emoji.
